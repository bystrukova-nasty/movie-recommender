# 🎬 Movie Recommender System

## Описание проекта

Movie Recommender System — это интеллектуальная система рекомендаций фильмов, построенная на основе алгоритмов коллаборативной фильтрации.

Проект использует оценки пользователей для прогнозирования интереса к фильмам и формирования персонализированных рекомендаций. В качестве рекомендательного алгоритма применяется метод матричной факторизации SVD (Singular Value Decomposition) из библиотеки Surprise.

Дополнительно разработан Telegram-бот, позволяющий получать рекомендации, искать фильмы, оценивать их и сохранять пользовательские предпочтения.

---

## Основные возможности

### 🎬 Рекомендации фильмов

* Генерация персональных рекомендаций.
* Вывод ТОП-10 наиболее рекомендуемых фильмов.
* Случайный фильм дня.

### 🔎 Поиск фильмов

* Поиск фильмов по названию.
* Отображение информации о найденных фильмах.

### ⭐ Система оценок

* Оценивание фильмов через Telegram-бота.
* Сохранение пользовательских оценок.
* Просмотр истории собственных оценок.

### 🤖 Telegram Bot

* Удобный интерфейс через кнопки.
* Персонализированные рекомендации.
* Хранение пользовательских предпочтений.

## Обучение модели

Для построения рекомендательной системы использовался алгоритм SVD из библиотеки Surprise.

В процессе работы были проведены эксперименты с различными гиперпараметрами модели.

Лучшая конфигурация:

* n_factors = 50
* n_epochs = 20

---

## Результаты

Качество модели оценивалось по метрике RMSE.

Полученный результат:

**RMSE = 0.877468**

Данный показатель демонстрирует хорошую точность предсказания пользовательских оценок.

## Запуск проекта

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск обучения модели

Открыть и выполнить ноутбук:

```text
Movie_Recommendation.ipynb
```

### 3. Запуск Telegram-бота

Установить переменную окружения:

```bash
BOT_TOKEN=YOUR_TELEGRAM_TOKEN
```

Запустить бота:

```bash
python bot/bot.py
```

## Пример работы системы

1. Пользователь запускает Telegram-бота.
2. Выполняет поиск фильма.
3. Ставит оценку через кнопки ⭐1–⭐5.
4. Оценки сохраняются в базе.
5. Система формирует персональные рекомендации на основе предпочтений пользователя.
